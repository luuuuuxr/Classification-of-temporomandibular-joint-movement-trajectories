# 改进的颞下颌关节疾病分类系统

## 🎯 主要改进

### 1. **基于原始轨迹的阶段分析**
- 使用运动学分析将轨迹分为6个阶段
- 提取每个阶段的统计特征和频域特征
- 确保分析结果有临床意义

### 2. **注意力机制分析**
- 使用带注意力机制的LSTM模型
- 分析模型关注的时间点
- 提供更直观的可解释性

### 3. **临床意义导向**
- 生成临床洞察报告
- 提供诊断建议
- 确保结果可指导临床实践

## 🚀 使用方法

### 运行改进版程序
```bash
python main_improved.py
```

### 程序流程
1. **数据加载**: 加载3D轨迹数据
2. **数据预处理**: 标准化和增强
3. **特征提取**: LSTM特征提取
4. **模型训练**: 多种分类器训练
5. **阶段分析**: 基于原始轨迹的阶段特征分析
6. **注意力分析**: 注意力机制可解释性分析
7. **报告生成**: 生成临床洞察报告

## 📊 输出结果

## �� 阶段特征说明

### 统计特征 (每个阶段29个特征)
- **位置特征**: 均值、标准差、方差、中位数、极差、分位数
- **运动特征**: 速度、加速度统计
- **形状特征**: 轨迹长度、直线距离、复杂度

### 频域特征 (每个阶段15个特征)
- **FFT特征**: 最大幅值、平均幅值、主频率、总能量

## 🏥 临床解释

### 6个运动阶段
1. **张口初始阶段**: 反映下颌开始运动的协调性
2. **张口过程阶段**: 反映张口过程中的运动稳定性
3. **张口结束阶段**: 反映最大张口时的关节位置
4. **闭口初始阶段**: 反映闭口开始时的运动控制
5. **闭口过程阶段**: 反映闭口过程中的运动协调
6. **闭口结束阶段**: 反映回到初始位置的运动精度

### 诊断建议
- 基于阶段重要性排序
- 提供针对性的检查建议
- 结合注意力分析结果
